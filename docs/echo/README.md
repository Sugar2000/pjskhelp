# ECHO命令速览

| 命令          | 示例             | 备注                            |
| ----------- | -------------- | ----------------------------- |
| bind `用户ID` | bind 123456789 | 绑定账号                          |
| echo        |                | 分数查询                          |
| mr          |                | 查看排位赛成绩                       |
| rank `排名`   | rank 10        | 获取指定排名的分数                     |
| target `排名` | target 100     | 设置追赶名次<br/>查询分数会同时返回目标名次并计算差距 |
| live        |                | 获取最近的3场live开始时间               |
| myid        |                | 查询目前绑定的id                     |
| 收歌进度        |                | 查询目前账号的歌曲完成情况                 |
